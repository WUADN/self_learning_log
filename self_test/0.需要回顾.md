### gcc 指令  
-c：仅编译源文件，生成目标文件（不进行链接）。  
-o <output_file>：指定输出文件的名称。  
-Wall：启用大多数警告消息。  
-Werror：将警告视为错误。  
-g：生成调试信息。  
-O<level>：指定优化级别（0 表示无优化，1 表示基本优化，2 和 3 表示更高级别的优化）。  
-I<path>：添加包含文件搜索路径。  
-L<path>：添加库文件搜索路径。  
-l<library>：链接指定的库文件。  
-std=<standard>：指定使用的 C 或 C++ 标准。
